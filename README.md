## Models
- Stable Diffusion v1.5
- Stable Diffusion XL
- Kandinsky 2.2

## How to use
1. Select a model in the sidebar
2. Enter your prompt + optional negative prompt
3. Adjust steps, CFG, and seed
4. Click "Generate"

Your image will appear on the main panel and can be downloaded.
